# PatentHub

This directory contains the core application code, managing both backend and frontend components.

## Commands

- `bun install`: Install all dependencies.
- `bun run dev`: Start backend and frontend concurrently.
- `bun run backend`: Start only the Hono server.
- `bun run frontend`: Start only the Vite server.

## Structure

- `/backend`: API services and business logic.
- `/frontend`: React application.

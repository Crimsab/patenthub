# PatentHub Frontend

Modern and responsive user interface for patent exploration, built with React, Vite, and Tailwind CSS.

## Features

- **Dark Mode Design**: UI optimized for technical reading with an elegant dark palette.
- **Split View**: Side-by-side view of the document and the AI assistant.
- **Real-time Search**: Integrated search suggestions and history.
- **RAG Status**: Visual indicators for document indexing state (Abstract vs Full Text).
- **Markdown Support**: Formatted AI analysis with GFM support.

## Tech Stack

- **UI Framework**: React 18
- **Styling**: Tailwind CSS + Lucide Icons
- **State Management**: React Hooks
- **Data Fetching**: Axios

## Development

To start the Vite development server:
```bash
bun run frontend
```

The app will be available at `http://localhost:5173`.

## Build

To generate production assets:
```bash
bun run build
```
